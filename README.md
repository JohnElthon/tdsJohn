# Projetos TDS - Senac Santos

Repositório com projetos e exercícios desenvolvidos durante o curso Técnico em Desenvolvimento de Sistemas no Senac Santos em 2024.

## Sobre

Aqui estão reunidos os projetos e exercícios feitos ao longo do curso, cobrindo HTML, CSS, JavaScript e PHP.

## Projetos em Destaque

### Projeto RPG - Dragon Ball
Site interativo com tema Dragon Ball desenvolvido em PHP com orientação a objetos. O usuário escolhe um personagem (herói ou vilão), e o sistema exibe seus atributos, ações básicas e ataques especiais com base em classes PHP.

- PHP com classes e herança
- HTML e CSS com tema personalizado
- 15 personagens jogáveis entre heróis e vilões

### Site MotoGP
Site de galeria dos fabricantes da MotoGP com menu lateral e cards de imagens.

- HTML e CSS
- Layout com flexbox

### Restaurante Peruano
Site de restaurante com header, galeria de pratos e seção sobre o restaurante.

- HTML e CSS
- Galeria de pratos com cards

## Exercícios de JavaScript

Vários exercícios práticos cobrindo manipulação do DOM, arrays, funções, operador ternário e eventos:

- **Menu Hamburguer** — menu responsivo que aparece/some conforme o tamanho da tela
- **Calculadora de Combustível** — calcula se vale mais abastecer com gasolina ou etanol
- **Horários do Dia** — exibe saudação e imagem de acordo com o horário atual
- **Lista Dinâmica** — lê números via prompt, ordena e exibe informações do array
- Outros exercícios de funções, DOM e eventos

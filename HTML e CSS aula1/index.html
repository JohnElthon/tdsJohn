Aula de HTML e CSS!

O HTML foi criado por Tim Berners-Lee em 1991 como uma linguagem de marcação para estruturar documentos na web.
O CSS (Cascading Style Sheets) foi desenvolvido posteriormente, com sua primeira especificação publicada em 1996,
para permitir a estilização e formatação de páginas HTML, separando conteúdo da apresentaçã Juntos,
HTML e CSS formam a base do desenvolvimento web
